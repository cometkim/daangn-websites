# websites

Daangn on the WWW

## Introduction

당근에서 공통으로 관리되는 웹사이트 소스코드 저장소입니다.

점점 빈번해지는 웹사이트 개발과 관리를 효율화 하고 모두가 기여할 수 있도록 맥락을 보존하는 모노레포입니다.
